# Claude fixture catalog

Every fixture in this directory was independently authored for PromptJuice from the binding
Claude `/usage` specification. No CodexBar or third-party code, fixtures, strings, tests,
comments, or assets were imported.

`manifest.json` is the authoritative inventory. The `Provenance/outcomes.json` paths are
deliberately fictional inputs used to test install-method classification. Files under `Live/`
are PromptJuice's sanitized real captures. Automated residual-privacy checks passed before
they entered the repository; their capture metadata is recorded in `Live/provenance.json`.
